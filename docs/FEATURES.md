# RUNNER — Features

## Core Loop

```
Préparer → Lancer → Focus 25min → Débrief → Archiver
```

---

## F1 — Préparation de Run (PREP Screen)

- Sélection ou création d'une **Quête** associée
- Sélection de la **Faction** (filtre visuel)
- Rédaction du **Contrat** (checklist de tâches)
- Note d'intention optionnelle (markdown court)
- Bouton "LAUNCH RUN"

---

## F2 — Run Active (ACTIVE Screen)

- Timer 25:00 décompte
- Affichage de la quête en cours
- Affichage du contrat (lecture seule pendant la run)
- Bouton "ABANDON RUN" (avec confirmation)
- UI minimaliste, immersive — inspirée HUD Marathon

---

## F3 — Débrief (DEBRIEF Screen)

- Quels contrats ont été cochés ?
- Note post-run libre (markdown)
- Rating rapide de la run (optionnel, ex: 1-3 étoiles ou icônes Marathon)
- Bouton "LOG RUN" pour archiver

---

## F4 — Quêtes

- Créer / modifier / archiver des quêtes
- Associer une faction à chaque quête
- Voir toutes les runs liées à une quête
- Notes attachées à la quête (style Obsidian — markdown plein, hiérarchie de blocs)
- Statut de progression visuel

---

## F5 — Notes (style Obsidian)

- Éditeur markdown natif
- Notes attachables à une quête ou à une run
- Liens internes entre notes `[[note-name]]`
- Vue liste des notes par quête / par date
- Pas de graphe de connaissance (hors scope v1)

---

## F6 — Journal de Bord (Historique)

- Liste de toutes les runs (date, durée, quête, statut)
- Filtrage par faction / quête / date
- Accès rapide au contrat + notes d'une run passée
- Statistiques simples : runs complétées, runs abandonnées, streak

---

## F7 — Dashboard (Quartier Général)

- Résumé du jour : runs effectuées, quêtes actives
- Bouton d'accès rapide "NOUVELLE RUN"
- Liste des quêtes en cours par faction
- Dernières notes

---

## Hors scope v1

- Sync cloud
- Multi-runner (collaboration)
- Audio ambiant
- Mobile
- Graphe de notes Obsidian
- Intégrations externes (GitHub, Notion, etc.)
